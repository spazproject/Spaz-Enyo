This example demonstrates the broadest theming solution. The box below contains three Buttons. Each button comes from a different package and expresses a different theme.
  • The unstyled button is CustomButton; it is the blank from which other buttons are derived.
  • The second button is an enyo.Button, which uses the default Heritage theme.
  • The third button is an Aristo.Button from the enyo-Aristo package.
  • The fourth button uses whatever theme was loaded at runtime.
