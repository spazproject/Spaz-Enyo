PDL_GetHardwareID

Description

Returns a constant for the current device the app is running on. These values are defined in PDL_types.h.

Syntax 

int PDL_GetHardwareID(void);

Returns 

HARDWARE_UNKNOWN    -1
HARDWARE_PRE        101
HARDWARE_PRE_PLUS   102
HARDWARE_PIXI       201
HARDWARE_VEER       301
HARDWARE_PRE_2      401
HARDWARE_PRE_3      501
HARDWARE_TOUCHPAD   601
