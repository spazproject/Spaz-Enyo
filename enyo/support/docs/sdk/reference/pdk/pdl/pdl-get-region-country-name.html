PDL_GetRegionCountryName

Description

Sets the passed buffer variable to the device's country/region stored in the internal luna preferences database. The passed buffer should be allocated enough memory to hold the anticipated value. The bufferLen parameter should be set to the length of the passed buffer.

Syntax 

PDL_Err PDL_GetRegionCountryName(char *buffer, int bufferLen);
Argument Description
buffer Pointer to buffer.
bufferLen Length of the buffer.

Returns 

PDL_INVALIDINPUT - buffer is null or zero length.
PDL_SYSTEMERROR_FILE - Unable to access database.
PDL_STRINGTOOSMALL - buffer is too small to hold value.
PDL_NOERROR - Success.
